# Campeões do Lol

## O que é

Esse jogo foi uma brincadeira que eu decidi fazer com alguns amigos enquanto decidia quais campeões são legais de se jogar e quais não.

## Como funciona 

Basicamente o jogo começa pedindo o nome de um campeão do lol e, dependendo de qual campeão você colocar, ele irá mostrar uma mensagem especial.
